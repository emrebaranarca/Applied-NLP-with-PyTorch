# Applied-NLP-with-PyTorch

## Overview

This repository contains a notebook-based NLP project focused on crisis detection from text data.
The main workflow is implemented in `Crisis-Detection-NLP-Pipeline.ipynb`.

## Project Structure

- `Crisis-Detection-NLP-Pipeline.ipynb`: End-to-end notebook for data loading, preprocessing, analysis, and model experiments.
- `LICENSE`: Apache License 2.0.

## Requirements

The notebook uses common data science and NLP libraries, including:

- Python 3.8+
- pandas
- numpy
- matplotlib
- scikit-learn
- PyTorch

Install the required packages in your environment before running the notebook:

```bash
pip install pandas numpy matplotlib scikit-learn torch
```

## Usage

1. Open `Crisis-Detection-NLP-Pipeline.ipynb` in Jupyter Notebook or JupyterLab.
2. Run the cells in order to reproduce preprocessing and modeling steps.
